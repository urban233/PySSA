Information

These are the official release notes of PySSA and the PySSA Installer. These release notes will open automatically if you first start a new PySSA version. In addition, the changelog can be accessed through the "Help" menu.

Features

What's new in v0.9.69

Rename proteins

  • It is now possible to rename protein structures from the edit page.

Ask to save pymol session

  • You can now decide, if you want a warning, to save your current pymol session should pop up or not.

Bug fixes

What's new in v0.9.69

  • The internal help of the Use page were falsely also on the View page.
  • Minor bug fixes.

Known issues

What's new in v0.9.69

  • Structure prediction does not work on every computer
  • The first structure prediction fails after updating the ColabFold ---Workaround---> Run the first prediction until it fails, and then you should be able to run your prediction like before the update.
  • Structure analysis fails if more than one analysis run is added to the job ---Workaround---> Run only jobs with one analysis run.

Support

If you run into any issues using PySSA, write an email or open an issue on GitHub. Be aware that we can only deliver support if a log file is available. The log files can be accessed through the "Settings" menu.
