Sequences Tab
=============

With the _Sequences_ tab you can ...

- :doc:`import<sequence_import>` existing .fasta files.
- :doc:`add<sequence_add>` protein sequence strings through copy & paste.
- :doc:`save<sequence_save>` stored sequences as .fasta files.
- :doc:`delete<sequence_delete>` stored sequences.
- rename stored sequences.
- display the sequence of a stored sequence.
