Getting Started
---------------

Delete a PyMOL scene
********************
1. Select a protein pair from the tree.
2. Click on |open_in_new_w200|, to load the pymol session of the selected protein pair.
3. Click on a scene in the tree you want to delete.
4. Click on |cancel_w200|, to delete the selected scene.
