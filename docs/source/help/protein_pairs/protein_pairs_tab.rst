Protein Pairs Tab
=================

With the Protein Pairs tab you can ...

- :doc:`delete<protein_pair_delete>` stored protein pairs.
- :doc:`load<protein_pair_load_session>` a pymol session to display the structure of a stored protein pair.
- :doc:`add<protein_pair_add_scene>` a pymol scene.
- :doc:`update<protein_pair_update_scene>` the selected pymol scene.
- :doc:`delete<protein_pair_delete_scene>` the selected pymol scene.
