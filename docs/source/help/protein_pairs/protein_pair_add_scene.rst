Getting Started
---------------

Add a new PyMOL scene
*********************
1. Select a protein pair from the tree.
2. Click on |open_in_new_w200|, to load the pymol session of the selected protein pair.
3. Click on |add_circle_w200|.
4. Enter a new scene name.
5. Click on **Add**.
