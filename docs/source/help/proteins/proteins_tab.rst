Protein Tab
===========

With the _Proteins_ tab you can ...

- :doc:`import<protein_import>` existing protein structures through .pdb files or by PDB id's.
- :doc:`save<protein_save>` stored proteins as .pdb files.
- :doc:`delete<protein_delete>` stored proteins.
- :doc:`load<protein_load_session>` a pymol session to display the structure of a stored protein.
- :doc:`add<protein_add_scene>` a pymol scene.
- :doc:`update<protein_update_scene>` the selected pymol scene.
- :doc:`delete<protein_delete_scene>` the selected pymol scene.
