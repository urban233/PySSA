Overview Sequences Tab

The Sequences tab stores all sequences of the project. Here you can import sequences from existing FASTA files or adding new sequences by copying and pasting the new sequence in the text box. For every sequence the name and the actual sequence can be displayed. The name of the sequence can also be changed. It is also possible to delete an existing sequence or save it as .fasta file.

How to display additional information about a sequence.

To display additional information about a sequence, select a sequence from the list. After clicking on the sequence the information will pop up in the table on the right side.

Support

If you run into any issues using PySSA, write an email or open an issue on GitHub. Be aware that we can only deliver support if a log file is available. The log files can be accessed through the "Settings" menu.
