Structure Analysis

This page aims to assist in the creation of an analysis job.


Analysis Job

Create

To create a structure analysis job, you have to add at least one structure analysis run.

To create a structure analysis run, click on Add to configure the run.

Select

First, you need to select for each combo box a protein structure you want to analyze.

After that, you select the chain(s) of protein structure 1 and then of protein structure 2.

IMPORTANT: You must select as many chains in structure 2 as you selected in structure 1.

Add

After that you added your structure analysis run to the list.

Images

Before you start, you can decide if during the analysis images should be created.

To enable the image creation, tick the checkbox "Create ray-traced images" (slow).

IMPORTANT: The image creation can take a long time, depending on your job configuration.

You can also create the images afterwards!
