Analysis Images

Here you can create an analysis images job which creates the images that would be created during the analysis.

Analysis images job

To create an analysis images job, you have to add at least one protein pair. To add a protein pair, click on a protein pair name in the list Protein structure analysis' and click on Add.

To remove a protein pair, click on a protein pair name in the list Protein structure analysis' for image creation and click on Remove.

To start the analysis images job, click on Start (Automatic image creation).
