Create new project

The purpose of this page is to assist in the creation of a new project.

Here you can create either an empty project or one with an existing protein.

To create a new project, type your project name in the text box.


Valid characters

The valid characters are: A-Z, a-z, 0-9, _ and -.


Add

To add an existing protein to your project, tick the checkbox Add existing protein.


PDB ID

You can either type the PDB ID or choose an existing .pdb file in your filesystem, by clicking on the tree dots.

To use a protein from the PDB, you need to have an internet connection!
