Edit proteins of current project

This page is for assistance with editing a project.


Delete

To delete a protein structure, click on a protein name and on Delete.


Clean

To clean a protein structure, click on a protein name from the list.

Then you can either clean the selected one (Update current protein structure) or

create a duplicate and clean that one (Create new protein structure).


Add

To add an existing protein to your project, click on the add button.

You can either type the PDB ID or choose an existing .pdb file in your filesystem, by clicking on the add button.

To use a protein from the PDB, you need to have an internet connection!


Save

To Save a .pdb file click on the save button.
