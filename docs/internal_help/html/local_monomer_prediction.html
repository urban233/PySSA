Local Monomer Prediction

Here you can create a prediction job for monomer proteins.

Prediction Job

To create a prediction job, you have to add at least one prediction run.

Prediction run

To create a prediction run, click on Add and type a new protein name in the text box. Then paste the sequence you want to predict into the text field and click on Add to add the run to the job.

Start job

To start the prediction job, click on Predict.

Note for experts (Colabfold)

If you don't want to use the force field or want to change the template mode you can click on Edit, to configurate this.
