Global Settings

This page provides assistance with operating the global settings.


Cycles (PyMOL)

It represents the highest possible quantity of cycles for rejecting outliers. It is advisable to change the cycles value if amino acid distances above a certain cutoff should be excluded from the RMSD calculation and distance analysis.

Minimum value

The minimum cycle value is 0.

Maximum value

The maximum cycle value is 99.

Default value

The default cycle value is 0.


Cutoff (PyMOL)

It represents the cutoff threshold for rejecting outliers during sequence alignment. The cutoff collaborates with the cycles to set a limit for excluding specific amino acid distances from the calculation of RMSD (which also includes the sequence alignment) and the distance analysis.

Moreover the cutoff value is neglected unless the number of cycles are more than 0!

Minimun value

The minimum cutoff value is 0.00.

Maximum value

The maximum cutoff value is 99.99.

Default value

The default cutoff value is 1.00.
