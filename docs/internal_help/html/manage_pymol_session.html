Manage PyMOL session

Here you can change specific settings which change the current PyMOL session.

To change the PyMOL session settings, select a protein or protein pair in the combo box.

Changeable settings

Color (PyMOL)

To change the color of the protein, select the color in the combo box.

Representation (PyMOL)

To change the representation of the selected protein, select the representation in the combo box.

Background color (PyMOL)

To change the background color of the session, select the color in the combo box.

Disulfid bonds (PyMOL)

To display the disulfid bonds, click on Show.
