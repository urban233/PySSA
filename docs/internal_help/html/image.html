Image

Here you can configure different options which impact the PyMOL session and the creation of an image.

You can also create and update PyMOL scenes on this page.

Image settings

Representation

defines how the protein or protein pair is displayed.

Background color

defines the background color of the current PyMOL session.

Renderer

defines which renderer should be used, needs to be defined if you want to make ray-tracing images.

Ray-Trace-Mode

defines how the protein or protein pair is displayed in the final image.

Ray Texture

defines what texture PyMol applies to the surface of your object during ray tracing.

Ray-Tracing

create images which are ray-traced.

Transparent Background

creates images with a transparent background. If ray-tracing is activated the background is automatically transparent.

PyMOL scenes

PyMOL scenes are used to store certain viewpoints for later reference.

Save (New scene)

saves a new PyMOL scene.

Update (Current scene)

updates the current PyMOL scene.

Images

Preview (Image)

creates a preview of the image, which would get created.

Save (Image)

creates the image and saves it as .png file.
