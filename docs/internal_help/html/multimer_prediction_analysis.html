Multimer Prediction + Analysis

Here you can create a prediction + analysis job for multimer proteins. This combines the prediction with a structure analysis afterwards.

Prediction + Analysis Job

To create a prediction + analysis job, you have to first create prediction runs and then structure analysis runs.

Prediction run

To create a prediction run, click on Add and type your protein name in the text box and add the sequences to your protein.

Analysis run

To create a structure analysis run, click on Go (To analysis setup). Then click on Add, and choose a protein for protein structure 1 and 2. After that, select the chain(s), first in protein structure 1 and then in 2. You have to select the same number of chains in both protein structures. Then click on Next, to add the run to the job.

Image creation

To enable the image creation, tick the checkbox Create ray-traced images (slow). IMPORTANT: The image creation can take a long time, depending on your job configuration. You can also create the images afterwards!

Start job

To start the prediction + analysis job, click on Start.

Note

For a step-by-step explanation, consult the tutorial which is accessable through the Help menu within the PySSA plugin.
