Use existing project

This page provides assistance for using a project.

Here you can create a new project based on the current one (use).


Use

To use the current project, type a new project name in the text box.

Then add the proteins you want in your new project to the list Protein structures in new project and click on Create (New project).


Add

To add a protein to the list, click on a protein name in the "Available protein structures" list and click on Add.


Remove

To remove a protein from the list, click on a protein name in the list "Protein structures" in new project and click on Remove.
